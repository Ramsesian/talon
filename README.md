### Welcome!
These are my talon user files. I use talon on Pop!_OS so some commands may or may not work depending on what you're running.

### Command Name Guide
**Common Commands**
- go [location]: commands that take you to the location
- [thing] switch: commands that toggle the thing on and off
- mark [bookmark/file path]: commands that take you to a bookmark or filepath
- insert [text]: commands the insert or past a texture object into a document
- text [markup]: a command that makes the selected text bold, italic, underline, etc.

**Common Keywords**
- term: terminal
- yank: copy
- location: URL
